# 🍕 Site de Pizzaria

Projeto desenvolvido com o objetivo de praticar desenvolvimento web utilizando o Visual Studio Code.

## 📌 Funcionalidades

- Página principal para clientes
- Visualização de produtos (pizzas)
- Interface simples e intuitiva
- Página administrativa (ADM) para gerenciamento de pedidos
- Separação de acesso entre usuário comum e administrador

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 🎯 Objetivo

Este projeto foi criado como prática após estudos de desenvolvimento, com foco em aplicar conceitos de criação de interfaces e organização de um sistema simples com área administrativa.
